# Proyecto-datax

#Este proyecto analiza un conjunto de datos sobre negocios verdes en Colombia.

# Objetivo
Identificar patrones en la distribución de negocios por región y tipo de producto.

## Herramientas utilizadas
- Python
- Pandas
- Matplotlib

## Análisis realizado
- Limpieza de datos
- Eliminación de duplicados
- Análisis exploratorio
- Visualización de datos

## Resultados
Se identificó una concentración de negocios en ciertos productos y regiones, 
evidenciando patrones en el sector de negocios verdes.
